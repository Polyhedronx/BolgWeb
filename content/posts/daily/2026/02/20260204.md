## 1. Rust 中的调试信息

```rust
// 必须为结构体显式选择打印调试信息功能
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}
fn main() {
    let rect1 = Rectangle {
        width: 30,
        height: 50,
    };
	// {:?} 打印调试信息，{:#?} 完美格式化打印
    println!("rect1 is {rect1:?}");
}
```

也可以使用 `dbg!` 宏来打印：

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}
fn main() {
    let scale = 2;
    let rect1 = Rectangle {
        // 因为 dbg! 返回表达式的值的所有权，所以 width 字段将获得相同的值
        width: dbg!(30 * scale),
        height: 50,
    };
	// 不希望 dbg! 拥有 rect1 的所有权，所以在下一次调用 dbg! 时传递一个引用
    dbg!(&rect1);
}
```

此时的输出结果为：

```shell
$ cargo run
   Compiling rectangles v0.1.0 (file:///projects/rectangles)
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.61s

[src/main.rs:10:16] 30 * scale = 60
[src/main.rs:14:5] &rect1 = Rectangle {
    width: 60,
    height: 50,
}
```
