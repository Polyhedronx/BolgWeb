## 1. 返回 NULL 的特殊处理方法

```postgresql
-- ()中的查询在未查找到数据时会返回 NULL，而回到外层 SELECT NULL 会直接返回 NULL。
SELECT (
    SELECT num
    FROM MyNumbers
    GROUP BY num
    HAVING COUNT(num) = 1
    ORDER BY num DESC
    LIMIT 1
) AS num
```

## 2. 查找第十行

```bash
# 其中 -n+10 表示从第十行开始读
# head -1 表示只读一行
tail -n+10 file.txt | head -1
```

## 3. 运用正则表达式在 Bash 中匹配电话号码

```bash
# -E 是普适性最高的正则形式
# ^ 用于匹配行首，$ 用于匹配行尾，以防类似 0(000) 000-0000 这种类型的数据也被匹配到
# 注意空格在正则中也会被匹配，所以不能随意缩进来美化代码
# 括号之类的特殊字符需要转义，类似 \()
grep -E '^([0-9]{3}-[0-9]{3}-[0-9]{4}|\([0-9]{3}\) [0-9]{3}-[0-9]{4})$' file.txt
```
