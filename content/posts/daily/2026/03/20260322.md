## 1. 前缀和的灵活用法

在写 [560. 和为 K 的子数组 ](https://leetcode.cn/problems/subarray-sum-equals-k/) 的时候，思来想去没找到时间复杂度合适的方法，于是问了问 AI，学习到了一种特殊的解题技巧：构造前缀和哈希表。

也就是将前 i 个数字之和储存到哈希表中，此时，对于任何一个 sum，只要找到哈希表中 sum - k 相应的键中出现的值的次数（之所以是次数而不是直接 +1，是因为**整数数组可能出现负数，此时就可能出现多次**。而如果题目是正数数组，就可以简单 +1）并累加进结果 res 就好了。

```rust
use std::collections::HashMap;

impl Solution {
    pub fn subarray_sum(nums: Vec<i32>, k: i32) -> i32 {
        // 这里用 with_capacity 优化空间
        let mut prefix = HashMap::with_capacity(nums.len());
        let mut sum = 0;
        let mut res = 0;

        // 初始化前缀和 0 出现 1 次，处理从数组开头到当前位置的子数组
        prefix.insert(0, 1);
        for num in nums {
            sum += num;
            let target = sum - k;
            if let Some(&count) = prefix.get(&target) {
                res += count;
            }
            // 解引用得到哈希表的值
            *prefix.entry(sum).or_insert(0) += 1;
        }

        res
    }
}
```
