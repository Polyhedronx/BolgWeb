## 1. Cpp 中两种自增的区别

由于 leetcode 上的链表题无法使用 Rust 写题解，所以重拾以前学过一点皮毛的 Cpp，遇到了一些小问题，比如：

> `i++` 和 `++i` 到底有什么区别？

简单而言，`i++` 是先把 `i` 的值给出去，然后再自增，代码示例和输出如下：

```c++
int i = 1;
int a = i++;
printf("i 的值为 %d，a 的值为 %d",i, a);

// 输出为 i 的值为 2，a 的值为 1
```

而对于 `++i` 则是先自增再将值给出去，代码示例和输出如下：

```c++
int i = 1;
int a = ++i;
printf("i 的值为 %d，a 的值为 %d",i, a);

// 输出为 i 的值为 2，a 的值为 2
```

注意，前者返回的是不可修改的自增前的临时副本（有一定开销），后者返回的是可修改的对变量本身的引用：

```c++
int i = 1;
++i = 10;	// 合法行：返回i的引用，直接修改i
i++ = 10;	// 报错行：返回的是临时值，无法赋值

printf("i = %d", i);	// 删去报错行之后会输出 10
```

所以最好是使用 `++i` 。

>  类似 Rust 一样直接写 `i += 1` 可不可以呢？

对于 Cpp，迭代器内部是不一定允许出现 `+=` 这样的算数运算符的，只能使用 `++` 这样的专用自增运算符。

## 2. 链表题的一些理解总结 1

### 基础

- 依据边界条件剪枝是很重要的，最简单的剪枝类似链表为空或者链表只有一个节点这种，一个 `if (head == nullptr || head->next == nullptr)` 的条件判断就能解决。
- 把一个节点当成普通的结构体来看就行了，不要被 `node->val`、`node->next` 这种写法唬住，其实就是一个是储存值的变量，一个是储存指向下一个节点的地址的变量。
- 使用临时变量储存指针是常用的，类似反转链表里就需要暂时断开链表的连接，此时可以依靠储存的指针续上：`ListNode* nextTemp = curr->next;` 后面就可以修改 `curr->next` 这个变量指向的地址值了。

### 技巧

- 下面的 [160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/) 的题解，就是运用了遍历两个链表的长度和一致的思想。

```c++
class Solution {
public:
    ListNode* getIntersectionNode(ListNode* headA, ListNode* headB) {
        if (headA == nullptr || headB == nullptr) {
            return nullptr;
        }
        ListNode *pA = headA, *pB = headB;
        while (pA != pB) {
            pA = pA == nullptr ? headB : pA->next;
            pB = pB == nullptr ? headA : pB->next;
        }
        return pA;
    }
};
```

- 快慢指针也很常用，速度差可以用来找中点（也能用来找任何一个等分点），还能通过追上的速度判断环形链表的长度，下面是我写的 [234. 回文链表](https://leetcode.cn/problems/palindrome-linked-list/) 的题解：

```c++
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        // 剪枝：链表为空或只有一个元素
        if (head == nullptr || head->next == nullptr) {
            return true;
        }
        // 快慢指针找中点，从 slow->next 开始就是第二段
        ListNode* fast = head;
        ListNode* slow = head;
        // 注意这里的判断顺序不能颠倒
        // 在 fast-> next 为空的前提下先访问 fast->next->next 会崩溃
        while (fast->next != nullptr && fast->next->next != nullptr) {
            fast = fast->next->next;
            slow = slow->next;
        }
        // 反转第二段链表，并找到两段需要比较的链表头
        ListNode* secondHalf = reverseList(slow->next);
        ListNode* p1 = head;
        ListNode* p2 = secondHalf;
        bool res = true;
        // 逐个节点比较数据
        while (p2 != nullptr) {
            if (p1->val != p2->val) {
                res = false;
                break;
            }
            p1 = p1->next;
            p2 = p2->next;
        }
        // 恢复链表
        slow->next = reverseList(secondHalf);
        return res;
    }

    ListNode* reverseList(ListNode* head) {
        ListNode* prev = nullptr;
        ListNode* curr = head;
        while (curr) {
            ListNode* next = curr->next;
            curr->next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    }
};
```
