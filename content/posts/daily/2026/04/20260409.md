## 1. uv 的一些使用技巧

### 基础操作

```bash
# 创建虚拟环境
uv venv

# 激活虚拟环境
.\.venv\Scripts\Activate.ps1

# 退出虚拟环境
deactivate

# 添加依赖
uv add <package_name>

# 运行程序
uv run main.py
```

---

### 内联元数据脚本

在脚本顶部添加 `/// script` 块声明依赖：

```python
# /// script
# dependencies = [
#   "requests>=2.32.0",
#   "pandas>=2.2.0",
# ]
# ///
import requests

response = requests.get("https://httpbin.org/get")
print(f"状态码：{response.status_code}")
```

运行方式：
```bash
uv run temp.py
```

---

### `--with` 临时指定依赖

脚本保持纯代码，无需修改：

```python
import requests
print(requests.get("https://httpbin.org/get").status_code)
```

运行方式：
```bash
# 单依赖
uv run --with requests temp.py

# 多依赖
uv run --with requests --with pandas temp.py

# 指定版本
uv run --with "requests>=2.32.0" temp.py
```

---

### 全局工具安装

将 CLI 工具永久安装到全局环境：

```bash
# 安装
uv tool install ruff

# 使用（全局可用）
ruff format temp.py
ruff check temp.py

# 管理
uv tool list            # 查看已安装工具
uv tool upgrade ruff    # 更新
uv tool uninstall ruff  # 卸载
```

---

### 环境隔离说明

- 脚本依赖自动隔离，不污染系统 Python
- 在有 `pyproject.toml` 的项目目录运行脚本时，需加 `--no-project`：
  ```bash
  uv run --no-project --with requests temp.py
  ```
- uv 自带缓存自动过期机制，长期不用的临时环境会被自动清理

---

### 缓存清理

```bash
# 清理所有缓存（推荐）
uv cache clean

# 精准清理指定脚本的临时环境
uv env list --script temp.py    # 先查看
uv env remove --script temp.py  # 再删除

# 完全恢复初始状态
uv tool uninstall --all
uv cache clean
```

> **提示**：不要手动删除 uv 的缓存目录，使用上述命令操作。

## 2. Morris 遍历在二叉树展开的应用

写 [114. 二叉树展开为链表](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/) 看到进阶的要求是空间复杂度为 $O(1)$，在二叉树里我目前只接触到 Morris 遍历是巧妙运用前驱指针来实现这种空间复杂度的，于是自然而然想着能不能用这种方法解决：

首先，我先找到左子树的最右下节点，看着示例图找找有什么特别的规律：

```text
     1
   /   \
  2     5
 / \     \
3   4     6
```

发现可以接上右子树，让 `4->5->6` 连接在一起：

```text
    1
   /
  2
 / \
3   4
     \
      5
       \
        6
```

然后把左子树接到右边，而根节点的左指针置空：

```text
1
 \
  2
 / \
3   4
     \
      5
       \
        6
```

接着让 `2`（也就是根节点的右节点）成为下一个根节点，重复上述步骤，就能完成展开。代码实现如下：

```c++
class Solution {
public:
    void flatten(TreeNode* root) {
        // Morris 遍历：
        // 1.先找到当前根节点的左子树的最右下节点
        // 2.将当前根节点的右子树连接到上述最右下节点的右指针
        // 3.将当前根节点的左子树连接到右指针处，同时将左指针置空
        TreeNode* curr = root;
        if (!curr) return;
        while (curr) {
            if (curr->left) {
                TreeNode* pre = curr->left;
                while (pre->right) {
                    pre = pre->right;
                }
                pre->right = curr->right;
                curr->right = curr->left;
                curr->left = nullptr;
            }
            curr = curr->right;
        }
    }
};
```
