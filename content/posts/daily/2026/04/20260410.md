## 1. 个人梯子搭建的简明指南

在如今互联网工具十分普及的当下，搭建一个个人的梯子并不是一件十分困难的事情。

- **优点**：相对于成千上万人共用的机场的节点来说，自己服务器的节点要纯净许多、网速也有保障，而且很难被墙。
- **缺点**：如果不使用真实的住宅 IP 替换个人购买的 Azure / 亚马逊云服务器的大厂静态 IP 的话，某些订阅和收付场景依然会被限制。但这个缺点对于常见的机场来说也存在。

下面我将总结一下我个人搭建梯子的过程，其中也会适当讲解一下预备知识：

1. **拥有自己的国外服务器**：类似 Azure / 亚马逊云之类的大型 VPS 提供商，都可以使用国内的付款方式，甚至通过学生认证免费拿到一定时间的服务器使用权限。学生认证一般是先通过 Github 的学生包认证，再在对应的供应商官网找到认证入口，如果认证出错或者没通过，还可以直接给客服发工单，一般会在 7 个工作日内解决。>>>>> 这一步会拿到你的服务器的 IP 地址和可用的用户 / 密码，是进入下一步的先决条件。

2. **通过连接工具登录服务器**：首先需要确保服务器放行了可用的 SSH 端口，一般默认放行 443 端口，需要在服务器的官网查看网络相关模块的详情。如果在意图形化界面的话，网络上也有很多好用的免费软件，例如 Xshell。但我偏向于使用终端的 SSH 功能直接连接，大概的操作就是在 Powershell / cmd 这样的终端里输入 `ssh username@ip`，比如 `ssh polyhedron@12:233:45:67` 之后就会提示输入对应的密码（如果在上一步里选择了密码登录）。登录进去之后可以选择把自己的常用电脑的公钥通过 `sftp` 上传到服务器中，然后简单配置一下 SSH 的放行规则，这里详情可以搜索搜索，反正不违法。>>>>> 这一步只要能登录服务器的终端环境就算成功。

3. **切换 root 用户并执行 v2ray 安装脚本**：使用 `sudo -i` 命令之后输用户密码，使终端进入 root 用户，然后输入 `bash <(wget -qO- -o- https://github.com/233boy/v2ray/raw/master/install.sh)`，回车后会自动执行，等待执行完毕会出现一串类似 `vmess:djishafdfabicnsdioahidsn...` 的链接和一个入站流量使用的端口号，比如 `48843`，之后需要回到服务器的官网去网络相关模块添加入站放行端口规则，其它都保持默认，只需要更改目标端口区域为你上面得到的端口号，然后添加。>>>>> 这一步需要添加入站放行端口规则，并拿到一串 vmess 开头的链接

4. **在需要使用梯子的设备上下载代理软件**：比如 Windows 可以下载 v2ray，iOS 可以下载 shadowrocket……只需要复制上一步所说的链接，打开代理软件，找到描述指向添加配置这种功能的项然后点击，一般都有读取系统剪贴板自动配置的功能。注意：在配置导入成功之后还应该找到类似 “系统代理配置” 这种描述的项，改为 “自动配置” 之类的，这是关键。

## 2. 从前序和中序遍历构造二叉树

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

```text
       3
     /   \
    9     20
         /  \
        15   7

输入: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出: [3,9,20,null,null,15,7]
```

这题我还真不会做，看了题解才知道原来是用前序找根节点，在根据前序找到的根节点用中序找左右子树的 `size`，然后回到前序根据中序找到的 `size` 向后递推找左子树的根节点和右子树的根节点……

这里有几个需要注意的问题：

1. 中序遍历的索引需要频繁地被**根据值查找**所以最好存进**哈希表**，不然时间复杂度会直接奔向 $O(n^2)$。前序遍历的索引只是频繁访问，直接使用数组的原生索引就行了，不需要存哈希。
2. 维护一个 `idx` 成员变量用来帮助在前序中递归找根节点：因为前序中所有元素的排列顺序都是 `根->左->右`，用这个指针挨个取根节点就行了。

```c++
class Solution {
public:
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        this->pre = preorder;
        // 将中序遍历数组的值和下标存入哈希表
        int len = inorder.size();
        for (int i = 0; i < len; ++i) {
            map[inorder[i]] = i;
        }
        return build(0, len - 1);
    }

private:
    unordered_map<int, int> map;
    int idx;
    vector<int> pre;

    TreeNode* build(int left, int right) {
        if (left > right) return nullptr;
        int rootVal = pre[idx++];
        TreeNode* root = new TreeNode(rootVal);
        // 递归左子树再递归右子树
        int mid = map[rootVal];
        root->left = build(left, mid - 1);
        root->right = build(mid + 1, right);
        return root;
    }
};
```
