## 1. 中级财务会计期中复习

### 一、所有者权益

#### 1. 定义与特征

- **定义**：狭义权益即所有者权益，公司制企业中又称股东权益，是企业资产扣除负债后，由所有者享有的**剩余净权益**，公式：`所有者权益=资产-负债`
- **广义权益**：包含债权人权益（企业负债）+ 所有者权益，对应企业两大核心资金来源。
- **特征**：无需偿还（除非减资、清算）、清算时劣后于债权清偿、凭借持股比例享有企业经营决策权与股利分配权、计量依赖资产和负债的计量，无单独专项准则规范。
- **权利**：股东大会投票表决权（最高决策权）、股利收益获取权。

#### 2. 完整构成体系

##### （1）投入资本类

|      项目       |                        定义与核算要点                        |                           会计处理                           |
| :-------------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
| 实收资本 / 股本 | 投资者按章程约定实际投入企业的资本金；股份制企业为股本，按股票面值核算，**面值仅为计量单位，核心是股东持股比例**（决定表决权与股利分配权）；出资方式包括货币、固定资产、存货、无形资产等 | 借：银行存款 / 固定资产等贷：股本 / 实收资本（面值 / 约定份额部分）贷：资本公积 - 股本 / 资本溢价（差额） |
|    资本公积     | 企业收到投资者超出注册资本 / 股本所占份额的投资，以及其他资本公积① 资本 / 股本溢价：最重要来源，投资者超额缴入的资本② 其他资本公积：权益结算股份支付等待期确认、权益法长投被投方其他权益变动等 | 用途：仅可用于转增资本，**不得用于弥补亏损**转增分录：借：资本公积 - 股本 / 资本溢价贷：股本 / 实收资本 |

##### （2）其他综合收益

- 定义：企业未在当期损益中确认的各项利得和损失，原参考文档中计入资本公积的可供出售金融资产公允价值变动，现单独列报。
- 分类：以后会计期间不能重分类进损益的其他综合收益、满足条件可重分类进损益的其他综合收益。

##### （3）留存收益类

企业从历年实现的利润中提取并留存于企业的内部积累，是所有者权益的内生来源，分为盈余公积和未分配利润。

1. **盈余公积**

   - 分类：法定盈余公积（公司法强制计提）、任意盈余公积（股东大会自主决定计提）。
   - 计提规则：以净利润为基础，抵减年初累计亏损后计提；法定盈余公积计提比例 10%，累计额达注册资本 50% 可不再计提。
   - 用途：弥补企业亏损、转增资本、特殊情况下发放现金股利 / 利润。
   - 关键限制：转增资本后，留存的法定盈余公积不得少于转增前注册资本的 25%。

2. **未分配利润**

   - 定义：企业历年滚存的、待分配或留待以后年度分配的利润，是利润分配流程完成后的剩余金额
   - 特点：未指定专门用途，企业自主支配权强；连年亏损会形成负数，抵减所有者权益总额。

#### 3. 业务全流程会计处理

##### （1）利润分配完整流程

1. 结转本年净利润

```
借：本年利润
贷：利润分配-未分配利润
```

2. 计提法定盈余公积

```
借：利润分配-提取法定盈余公积
贷：盈余公积-法定盈余公积
```

3. 计提任意盈余公积

```
借：利润分配-提取任意盈余公积
贷：盈余公积-任意盈余公积
```

4. 宣告分配现金股利

```
借：利润分配-应付现金股利
贷：应付股利
```

5. 期末结转利润分配明细科目

```
借：利润分配-未分配利润
贷：利润分配-提取法定盈余公积
    利润分配-提取任意盈余公积
    利润分配-应付现金股利
```

##### （2）盈余公积补亏处理

```
借：盈余公积
贷：利润分配-盈余公积补亏

借：利润分配-盈余公积补亏
贷：利润分配-未分配利润
```

#### 4. 高频考点与易混辨析

- 资本公积 vs 盈余公积：前者来自资本投入，与企业经营利润无关；后者来自净利润积累，属于经营成果留存。
- 实收资本 vs 注册资本：前者是企业实际收到的出资额，后者是章程约定的认缴出资额。
- 直接计入所有者权益的利得 / 损失：核心为其他综合收益，不计入当期损益，不影响企业净利润。

### 二、收入

#### 1. 定义与特征

- **定义**：企业在日常活动中形成的、会导致所有者权益增加的、与所有者投入资本无关的经济利益的**总流入**。

- **特征**：
  1. 产生于企业日常经营活动，**非日常活动形成的利得（如营业外收入）不属于收入。**
  2. 会导致所有者权益增加，且与所有者投入资本无关。
  3. 是经济利益总流入，而非净流入。
  
- **难点**：收入的确认与实现判断，是会计处理的争议点。

#### 2. 收入确认与计量：新收入准则五步法模型

1. **第一步：识别与客户订立的合同**

   合同成立需同时满足：合同各方已批准并承诺履约、明确各方权利义务、明确支付条款、合同具有商业实质、对价很可能收回。

2. **第二步：识别合同中的单项履约义务**

   履约义务：合同中企业向客户转让可明确区分商品的承诺，是收入分摊与确认的核心单元。

3. **第三步：确定交易价格**

   企业预期有权收取的对价金额，需考虑可变对价、重大融资成分、非现金对价、应付客户对价的影响。

4. **第四步：将交易价格分摊至各单项履约义务**

   分摊基础：各单项履约义务所承诺商品的单独售价相对比例。

5. **第五步：履行履约义务时确认收入**

   核心原则：客户取得相关商品控制权时确认收入；分为某一时段内履约（按履约进度确认）、某一时点履约（控制权转移时点确认）。

#### 3. 收入的分类

|      收入类型      |                          定义与场景                          |                     确认要点                     |
| :----------------: | :----------------------------------------------------------: | :----------------------------------------------: |
|    销售商品收入    | 企业通过销售产成品、商品、原材料等取得的收入，是制造业 / 商贸企业核心收入 |   某一时点确认收入判断商品控制权是否转移给客户   |
|    提供劳务收入    | 企业通过提供服务取得的收入，如律师服务、咨询服务、运输服务等，是第三产业核心收入形式 |  大多属于某一时段内履约义务，按履约进度确认收入  |
| 让渡资产使用权收入 | 企业将闲置资产让渡他人使用取得的收入，包括利息收入、无形资产 / 固定资产出租的使用费收入 | 相关经济利益很可能流入、收入金额能可靠计量时确认 |

#### 4. 特定交易会计处理

##### （1）售后回购业务

- 判断逻辑：不看合同形式，看商品控制权是否真正转移，本质是否为融资 / 租赁交易。

- 情形处理：

  1. 回购价≥原售价：属于融资交易，不确认收入；收到款项确认为金融负债，回购价与原售价差额确认为融资费用（财务费用）。
  2. 回购价＜原售价：属于租赁交易，按租赁准则核算。
  3. 回购价 = 原售价：视同销售退回，不确认收入。

##### （2）销售折扣、折让与退回

- 商业折扣：直接按扣除折扣后的金额确认收入，无需单独账务处理。
- 现金折扣：为鼓励购货方提前付款提供的折扣，实际发生时计入**财务费用**（融资性质支出）。
- 销售折让：因商品质量问题给予的价格减让，发生时冲减当期销售收入。
- 销售退回：商品质量问题导致的退货，发生时冲减当期销售收入，同步冲减对应销售成本。

##### （3）中级补充高频特定交易

附有销售退回条款的销售、附有质量保证条款的销售、主要责任人与代理人区分（总额法 vs 净额法）、附有客户额外购买选择权的销售（奖励积分）、知识产权许可、售后代管商品安排。

#### 5. 高频考点与易混辨析

- 收入 vs 利得：收入是日常活动形成的总流入，利得是非日常活动形成的净流入；营业外收入、资产处置损益不属于收入。
- 合同资产 vs 应收账款：合同资产是**有条件的收款权**，应收账款是**无条件的收款权。**
- 控制权转移 vs 风险报酬转移：现行准则以控制权转移为收入确认原则，替代原准则的风险报酬转移。

### 三、费用

#### 1. 定义与特征

- **定义**：企业在日常活动中发生的、会导致所有者权益减少的、与向所有者分配利润无关的经济利益的**总流出**。

- **特征**：
  1. 产生于企业日常经营活动，**非日常活动形成的损失（如营业外支出）不属于费用**。
  2. 会导致所有者权益减少，且与利润分配无关。
  3. 与收入形成配比关系，公式：`收入-费用=利润`
  
- 准则边界：会计准则中费用定义不包含损失，损失需单独核算。

#### 2. 概念辨析

| 概念 |                             定义                             |                          边界与区别                          |
| :--: | :----------------------------------------------------------: | :----------------------------------------------------------: |
| 费用 | 企业日常活动发生的、计入当期损益的经济利益总流出，狭义指期间费用 |   与日常活动相关，直接影响当期利润，范围小于支出、大于成本   |
| 成本 | 会计中特指历史成本 / 生产成本，是**对象化的费用**，包括直接材料、直接人工、制造费用 | 依附于特定产品 / 项目，未完工 / 未销售的成本不影响当期损益，是费用的子集 |
| 支出 | 企业经济利益的总流出，分为收益性支出（计入当期费用）、资本性支出（形成资产） | 范围最大，资本性支出先计入资产，后续通过折旧 / 摊销逐步计入费用 |
| 损失 |   企业非日常活动发生的、会导致所有者权益减少的经济利益流出   | 与费用区别：非日常活动形成；分为计入当期损益的损失（营业外支出）、计入所有者权益的损失 |

#### 3. 费用的分类与核算

##### （1）生产成本（对象化的费用）

- 定义：企业为生产产品、提供劳务发生的各项可归属于特定产品的耗费。

- 构成：

  1. 直接材料：直接用于产品生产、构成产品实体的原材料、辅助材料。
  2. 直接人工：直接从事产品生产的工人职工薪酬。
  3. 制造费用：生产车间发生的间接生产费用，如车间固定资产折旧、车间管理人员薪酬、水电费等。

##### （2）期间费用（狭义费用）

企业日常活动发生的、不能计入特定产品成本，直接计入当期损益的费用，是中级考试核心核算范围。

1. **销售费用**

   - 定义：一切与企业销售产品 / 服务相关的、为促进销售发生的费用。
   - 核算内容：广告费、业务宣传费、展览费、销售佣金、专设销售机构的各项支出、销售环节运输费 / 装卸费 / 包装费等。

2. **财务费用**

   - 定义：企业融通资金发生的全部融资费用，不仅限于借款利息。
   - 核算内容：利息支出（减利息收入）、汇兑损益、金融机构手续费、现金折扣、未确认融资费用摊销等融资相关支出。

3. **管理费用**

   - 定义：企业行政管理部门为组织和管理生产经营活动发生的各项费用，属于 “大杂烩” 科目。
   - 核算内容：筹建期间开办费、行政管理人员薪酬、办公费、差旅费、业务招待费、工会经费、董事会费、房产税 / 车船税 / 印花税等、无形资产摊销、研发费用费用化部分等。
   - 考点：管理费用的高低可直接反映企业管理成本与费用控制能力。

#### 4. 会计处理

1. 费用发生时

```
借：管理费用/销售费用/财务费用/生产成本
贷：银行存款/应付职工薪酬/累计折旧/原材料
```

2. 生产成本结转（产品完工入库）

```
借：库存商品
贷：生产成本
```

3. 期末结转期间费用损益

```
借：本年利润
贷：管理费用/销售费用/财务费用
```

#### 5. 高频考点与易混辨析

- 生产成本 vs 期间费用：前者对象化计入产品成本，销售后才影响损益；后者发生当期直接计入当期损益。
- 制造费用 vs 管理费用：前者是生产车间间接费用，计入产品生产成本；后者是行政管理部门费用，直接计入当期损益。
- 资本化支出 vs 费用化支出：资本化支出计入资产成本，分期影响损益；费用化支出发生当期直接计入损益，重要区分是支出的受益期长短。

### 四、三大要素联动逻辑

1. **资产负债表恒等式**：`资产 = 负债 + 所有者权益`

   所有者权益为剩余权益，其计量完全依赖资产和负债的计量；变动来源包括投资者投入 / 减资、净利润 / 净亏损、直接计入所有者权益的利得和损失、利润分配。

2. **利润表恒等式**：`收入 - 费用 = 利润`

   利润最终归属于企业所有者，期末净利润结转至未分配利润，成为所有者权益的组成部分，实现资产负债表与利润表的勾稽联动。

3. **关联**：所有者权益的内生驱动是企业盈利（收入 - 费用形成的净利润），三大要素共同构成企业财务状况与经营成果的核算体系，是中级财务会计的主线。
