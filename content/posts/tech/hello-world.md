## 欢迎来到我的博客

这是我的个人博客，使用以下技术栈构建：

- **前端**: React + TypeScript + shadcn/ui + Tailwind CSS
- **后端**: Go + Gin
- **数据库**: PostgreSQL
- **部署**: Docker Compose

## 为什么选择这个技术栈？

### Go + Gin

Go 语言编译迅速、运行效率高、部署简单（单个二进制文件）。Gin 是目前最流行的 Go Web 框架，性能优秀且生态完善。

### React + shadcn/ui

React 生态成熟，shadcn/ui 基于 Radix UI 提供了高质量的无障碍组件，搭配 Tailwind CSS 可以实现高度定制化的设计。

### PostgreSQL

世界上最先进的开源关系数据库，稳定可靠，功能丰富。

### Docker Compose

一条命令启动所有服务，开发和生产环境一致性有保障。

## 博客功能

- **Markdown 写作** — 所有文章以 `.md` 格式存储在仓库中
- **标签与分类** — 方便检索和组织内容
- **评论系统** — 支持嵌套回复
- **全文搜索** — 快速找到想要的内容
- **RSS 订阅** — 方便读者订阅更新
